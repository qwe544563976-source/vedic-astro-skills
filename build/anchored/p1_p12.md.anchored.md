## P1 角色身份 [关键权重]

行星的"职务"由它管理的宫位决定：

<!-- ANCHOR: A-P12-08744fb0 L2 -->
```
Core-Driver (1宫主) = 忠诚者
  → 永远服务盘主，力量越强越好

Yogakaraka (同掌吉凶宫) = CEO
  → 同时管三角宫和角宫，最高价值

Faithful (5/9宫主) = 盟友
  → 三角宫主，天然吉利

Trader (2/4/7/10宫主) = 交易者
  → Kendradhipati法则：吉星管角宫→变怠工，凶星管角宫→变自律

Growth-Hacker (3/6/11宫主) = 竞争者
  → 带毒增长，力量越强副作用越大

Destroyer (8/12宫主) = 清理者
  → 擅长结束和清理，科研/审计/海外有价值
```

**复合身份（面向用户的翻译）**：
- Soft-Betrayal = "温和错位"（吉星担凶职）→ 好心但用错了地方
- Hard-Construction = "高压建设者"（凶星担吉职）→ 苦劳出成果
- Growth-Hacker = "成长型竞争者" → 越用越强但有内耗

<!-- ANCHOR: A-P12-c45018cc L2 -->

```
  禁止：美化为"转化的力量""深层的智慧"

  触发条件：自然凶星（Saturn/Mars/Rahu/Ketu）担任Core-Driver或Yogakaraka角色
```

### P2.5 Moon 盈亏合成规则（Moon 专项，防系统性方向错）

<!-- ANCHOR: A-P12-273aeac1 L2 -->
```

  外显度：火/风象 Moon = 外露直接（说话直/情绪外显），土/水象 = 内收沉潜
  盈月 = 深度/续航强（情绪有后劲、记得久）

  正确 = "火性外显但后劲不足"（直、急、外露、来得快去得快）——完全反向。
涉及母亲画像/DK配偶画像/心意底色/Moon 参与的 Dasha 时期，全部按本规则合成。
```

## Karaka 使用铁律：符号锚点 ≠ 写实剧本

<!-- ANCHOR: A-P12-f37bb91d L2 -->
```
Karaka（BK/MK/DK/AK/AmK…）是"某个人生领域的符号锚点"——它说的是
该领域能量场的强度与方向，不是那个领域里具体人物与盘主的心理关系剧本。

允许推（能量结构层）：
  ✅ 行星位置/尊贵度/相位（数学层）
  ✅ 该领域人物"本身的能量场特征"（如"父亲能量场活跃/在家庭里份量重"）
  ✅ 原生家庭/关系氛围的客观元素（经济/精神/结构氛围倾向）

禁止推（心理写实剧本——反复出错的方向）：
  ❌ "你和父亲/母亲的心理距离/情感关系如何"
  ❌ "父母对你的塑造/影响/上帝视角"
  ❌ "你会下意识想到TA怎么看/你内心觉得X"这类断言式心理事实
  ❌ 把"能量场强"翻译成"心理关系密切/塑造力强"

核心区分：**TA的能量场什么样 ≠ 你和TA的相处/心理关系什么样**——
前者盘面能说，后者盘面不直接说。写报告涉及此层时，措辞必须软化为
"能量场结构参考"并明示"你的直接感受优先"；共鸣式表达可用
（"如果你也有这种感觉，它对应盘面的X"），断言式禁止。

分层可信度（写作与QA引用时对照）：
  数学层（尊贵度/宫主/SAV/Dasha时间线）= 高
  能量结构层（领域方向/强度倾向）= 中高
  时间节点（大运时点数学确定，能量方向是倾向非剧本）= 中高
  关系/心理写实层 = 低——用户直接感受优先，不符合就是不符合
```
