# 格局(Yoga)评估方法论

<!-- ANCHOR: A-YG-96c753a0 L2 -->
> 用途：core在十大板块第10项"格局评估"时参考
> 原则：说潜力和突破口，不定命
> ⚠️ **格局"互视 / 互溶"一律读 structured_data「格局关系类 · 互视 / 互溶」段判定**（calc 已算好、❌禁手推、❌禁先验）：
>   · **互视 = 双向 mutual**（A照B 且 B照A，calc 已判、单向照射已排除）——读「互视对」表；**宫主互视只取行星对，节点 Rahu/Ketu 不掌宫、不参与格局**。❌禁自己从 Graha Drishti 表判双向、❌禁用西占 orb 相位（原「主要相位关系」表已废弃删除）。单向照射（只一方照）= 弱信号，不成格局（如 Dharma-Karma 只有 9L 单向照 10L → 不成立）。
>   · **互溶 = parivartana**（两宫主互换落座，calc 已判、含 Maha/Khala/Dainya 分类）——读「互溶对」表，❌禁自推互换。
> ⚠️ **合相 = 同 rashi 同宫**（看落宫/同星座；度数强弱分档唯一口径见 p1_p12「合相度数三档」）——❌不是西占跨座 orb 角（合相同宫太简单、未下沉，看落宫号即可）。

## 主要格局扫描清单

<!-- ANCHOR: A-YG-ed82e30a L1 -->
### Dharma-Karma Yoga（使命格局）
<!-- ANCHOR: A-YG-a5d313ee L2 -->
```
条件：9宫主和10宫主互动（合相/互视/互溶）
含义：事业有使命感，社会认可度高
评估：
  参与星强 → "你的事业天然带有意义感，容易获得社会尊重"
  参与星弱 → "有这个潜力但需要特定Dasha时期才能激活"
```

<!-- ANCHOR: A-YG-2703c00c L1 -->
### Dhana Yoga（财富格局）
<!-- ANCHOR: A-YG-d4e168f9 L2 -->
```
条件：2宫主和11宫主互动（合相/互视/互溶）
      或 2/11宫主和1/5/9宫主互动（合相/互视/互溶）
含义：财富积累能力强
评估：
  看参与星的Shadbala和尊贵度（主判据）
  2宫和11宫的SAV 仅极端值(>32/<20)时作资源佐证，❌不与 Shadbala/尊贵度并列为判据
```

<!-- ANCHOR: A-YG-55d3555a L1 -->
### Raja Yoga（王者格局）
<!-- ANCHOR: A-YG-a3c5a854 L2 -->
```
条件：三角宫主(1/5/9) + 角宫主(1/4/7/10) 合相/互视/互溶
      （互溶也成立——两宫主互换管辖 = 双向激活，等同 Raja Yoga 通道）
含义：权力/地位/影响力
评估：
  纯度 = 参与星是否同时管凶宫（如果是则打折）
  强度 = 参与星的Shadbala
```

<!-- ANCHOR: A-YG-97b7f1a2 L1 -->
### Viparita Raja Yoga（逆境王格局）
<!-- ANCHOR: A-YG-6529eda4 L2 -->
```
条件：6/8/12宫主落入其他凶宫(6/8/12)——唯一判定定义见 p1_p12.md P5（落宫式；两凶宫主互溶=最强特例）。**结构成立读 structured_data「结构格局预扫·VRY」表 active 字段，禁自推**
含义：在困难中崛起，从别人的损失中获益
评估：
  存在 → "你有一种特殊能力：越是困难的环境，你越能找到出路"
  注意：不要过度美化，VRY的代价是必须经历逆境
```

<!-- ANCHOR: A-YG-6a24fe97 L1 -->
### Gajakesari Yoga（象王格局）
<!-- ANCHOR: A-YG-5d057ea5 L2 -->
```
条件：Jupiter和Moon互为角宫（相距1/4/7/10宫）
含义：智慧+情感稳定，学习能力强
评估：
  Jupiter和Moon的Shadbala决定实现程度
```

<!-- ANCHOR: A-YG-b0def53f L1 -->
### Adhi Yoga（辅弼格局）
<!-- ANCHOR: A-YG-7129bfed L2 -->
```
条件：自然吉星（Jupiter/Venus/Mercury）落在从Moon起第6/7/8宫，且所落宫无凶星混杂
  ⚠️ 判据与Gajakesari（Jupiter/Moon互为角宫）不同，禁止混同；
     "Moon旁边有吉星"不构成本格局
含义：领导力/权威/辅佐得力，地位与庇护有支撑
评估：
  6/7/8三宫均有吉星且无凶星 → 完整成立，效力最强
  仅1-2宫有吉星 → 部分成立，效力递减
  所落宫混有凶星 → 减损，标注"弱成立"
  参与吉星燃烧/落陷/弱Shadbala → 按P12条件修正打折
```

### Chandra-Mangala Yoga（月火格局）
<!-- ANCHOR: A-YG-af114c29 L2 -->
```
条件：Moon和Mars合相或互视
含义：行动力+直觉力结合，适合创业
评估：
  吉利组合但有情绪波动的副作用
```

<!-- ANCHOR: A-YG-0eb63160 L1 -->
### Kemadruma Yoga（月亮孤独格局）
<!-- ANCHOR: A-YG-4cc9d8d0 L2 -->
```
条件：Moon前后两宫(从Moon算2宫和12宫)均无行星
含义：情绪孤立/心理孤独感，内心缺乏安全感
评估：
  成立 → "你的内心容易感到孤独，需要主动建立情感支持系统"
  
  解除条件（任一满足即解除）：
    - Moon在Kendra(1/4/7/10宫)
    - Moon有行星合相
    - Moon受Jupiter相位
    
  解除 → "Kemadruma被缓解，孤独感减弱但底色仍在"
```

<!-- ANCHOR: A-YG-bdc25787 L1 -->
### Pancha Mahapurusha Yoga（五大贤者格局）
<!-- ANCHOR: A-YG-e7f0858f L2 -->
```
条件：Mars/Mercury/Jupiter/Venus/Saturn 落在Kendra(1/4/7/10) 且 入旺或入庙
  → 5颗星各形成一个独立格局：

  Ruchaka (Mars in Kendra + 入旺Capricorn/入庙Aries或Scorpio)
    → 勇武、领导力、体魄强健、军事/执法/工程才能

  Bhadra (Mercury in Kendra + 入旺Virgo/入庙Gemini)
    → 智慧、商业才能、沟通/写作/教育天赋

  Hamsa (Jupiter in Kendra + 入旺Cancer/入庙Sagittarius或Pisces)
    → 学识、精神智慧、道德感、教育/学术/法律

  Malavya (Venus in Kendra + 入旺Pisces/入庙Taurus或Libra)
    → 艺术鉴赏、人际魅力、奢华享受、美学/设计/外交

  Shasha (Saturn in Kendra + 入旺Libra/入庙Capricorn或Aquarius)
    → 权威、纪律、管理才能、政治/行政/建筑

评估：
  参与星燃烧/逆行/战争 → 格局打折
  参与星受凶星相位 → 效力减弱
  成立 → "你有[格局名]，在[领域]有先天的高起点"
```

<!-- ANCHOR: A-YG-3f09036d L1 -->
### Guru Chandala Yoga（木罗污染格局）
<!-- ANCHOR: A-YG-0c548117 L2 -->
```
条件：Jupiter和Rahu合相（同宫，距离<15°效力更强）
含义：Jupiter的智慧/道德判断被Rahu放大扭曲
  → 过度自信、判断失误、非正统手段获取成功
评估：
  Jupiter强+Rahu弱 → 污染较轻，"偶尔判断失准但大方向不偏"
  Jupiter弱+Rahu强 → 污染严重，"容易被欲望和捷径诱惑"
  Jupiter入旺/入庙 → 抵抗力强，Guru Chandala被压制
  成立 → "你的直觉和判断力有时会被'贪心'或'走捷径'的念头干扰，
          重大决策建议多听第三方意见"

  ⚠️ 不要过度恐吓——很多成功人士盘里有Guru Chandala，
     它也代表非传统路径的成功（创业/创新/跨界）
```

<!-- ANCHOR: A-YG-cf1a052f L1 -->
### Shakata Yoga（月木车轮格局）
<!-- ANCHOR: A-YG-19856549 L2 -->
```
条件：Jupiter和Moon互为6宫或8宫关系（从Moon数Jupiter在6/8宫，或反之）
含义：情绪稳定性和智慧之间的周期性脱节
  → 好运不持久，反复起落（像车轮一样转动）
评估：
  解除条件（任一满足即解除）：
    - Jupiter在Kendra(1/4/7/10宫，从Lagna算)
    - Moon在Kendra(从Lagna算)
    - Jupiter入旺/入庙

  成立且未解除 → "你的运势容易出现周期性波动，好的时候特别好，
                  但不容易守住成果。建议在顺境时多储备"
  成立但已解除 → "Shakata被化解，波动底色仍在但影响可控"
```

<!-- ANCHOR: A-YG-70ca0a54 L1 -->
### Parivartana Yoga（互溶格局）
<!-- ANCHOR: A-YG-79b28537 L2 -->
```
条件：两颗宫主互换落座——A星管X宫落在Y宫，B星管Y宫落在X宫
含义：X宫和Y宫的事务被强制捆绑，形成双向通道
分类（按参与宫位定吉凶）：
  - Maha Parivartana（吉宫互溶：1/2/4/5/7/9/10/11 之间互换）→ 双赢，两领域互相成就
  - Khala Parivartana（3宫主参与的互溶）→ 努力后有收获，过程波折
  - Dainya Parivartana（凶宫互溶：6/8/12 之间互换）→ 困境捆绑，福祸相依
评估：
  参与的两颗星的 Shadbala 和尊贵度决定捆绑质量
  ⚠️ 互溶本身可构成其他格局，扫格局时必须并入对应判定、不能只当独立条目漏掉：
     互溶双方是 角宫主+三角宫主 → 等同 Raja Yoga 通道；
     是 2/11 宫主 → 等同 Dhana Yoga 通道；
     是 9/10 宫主 → 等同 Dharma-Karma 通道。
```

## 评估框架

<!-- ANCHOR: A-YG-5cdf296f L2 -->
```
Step 1: 扫描上述格局是否存在
Step 2: 评估参与星状态

  格局存在 + 参与星强(Shadbala>120%) = 高实现概率
  → "这个格局在你盘中是活跃的，已经或即将体现在生活中"

  格局存在 + 参与星弱(Shadbala<100%) = 低实现但有潜力
  → "这个格局存在但需要适当的时机（通常是参与星的Dasha）才能激活"

  格局存在 + 参与星受损(燃烧/落陷) = 有格局但增益×0.2
  → "有这个倾向但实现起来会遇到阻力"

  格局不存在 = 看功能性等价物
  → 没有正式的Raja Yoga，但如果10宫主很强（落宫吉+Shadbala高+接通事业Yoga通道），
     事业方面依然有很好的基础；SAV>32 仅作强度加注，❌不单独顶替/成立准格局

Step 3: 标注突破口
  → 什么时期最可能兑现？（参与星的Dasha时期）
  → 什么领域最可能突破？（参与宫位对应的领域）
```

## 赛道优势合成

<!-- ANCHOR: A-YG-3203548d L2 -->
```
基于Step 1-3的结果，按四维框架输出：

a) 加持赛道 — 格局命中的领域
   多个Raja Yoga → 管理/权力赛道有先天加持
   Dhana Yoga强 → 财富积累赛道有加持
   AK/AmK力量强 → 事业/灵魂使命赛道有加持
   Gajakesari → 教育/学术赛道有加持

b) 经营赛道 — 格局缺失但可后天发展的领域
   无正式格局但功能性等价物存在（10宫主强=落宫吉+Shadbala高+Yoga通道；SAV>32 仅强度加注、不单独成立）
   → 说"需要后天经营，关键在[具体宫位/行星]"

c) 财富路径 — 钱从哪条路来
   Dhana Yoga + 2宫主强 → 工资/产品型收入（2宫SAV>32 时作强度佐证，不单独定路径）
   11宫主强 + 11宫格局 → 人脉/流量型收入（11宫SAV>32 时作强度佐证，不单独定路径）
   8宫强 + VRY → 他人资源/投资/佣金型
   D4交叉 → 不动产/实物资产潜力

d) 兑现窗口 — 格局在哪个大运激活
   参与星的Dasha时期 = 格局兑现窗口
   → "你的[格局名]在[行星]大运（[年份]-[年份]）最可能兑现"

禁止表达：
  ❌ "你的层次是高/中/低"
  ❌ "你命中注定会..."
  ❌ "你的命很差/很好"

正确表达：
  ✅ "你在[领域]赛道有[格局名]加持，
     在[Dasha]时期最可能兑现"
  ✅ "财富主要通过[路径]积累，
     [时期]是关键积累期"
  ✅ "[领域]没有格局加持，
     但[宫位/行星]状况不错，后天经营有空间"
```
