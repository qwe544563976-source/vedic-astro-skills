## 1. 宫位速查（Parashari 义）

| 宫 | 主要含义 |
|---|---|
| 1 | 自身/身体/事情本身；提问者的立场 |
| 2 | 财富/家庭/言语/累积资产 |
| 3 | 兄弟姐妹/勇气/主动努力/短途 |
| 4 | 母亲/家/房产/车/心灵安宁 |
| 5 | 子女/学业/恋爱/投机/创意/前世功德 |
| 6 | 疾病/敌人/诉讼/债务/下属/日常琐碎 |
| 7 | 配偶/伴侣/合作/公开对手/商业交易 |
| 8 | 突变/遗产/慢性病/性/隐秘/研究/寿命 |
| 9 | 父亲/法律/宗教/长途/运气/上师 |
| 10 | 事业/职位/公众声望/权威 |
| 11 | 收益/朋友/大目标/兄长/贵人/愿望达成 |
| 12 | 支出/损失/外国/隔离/精神追求/床笫/囚禁 |

**Dushtana(阻碍宫)**：3 / 6 / 8 / 12。相关宫主落此 = 事在阻碍位（除非有 Raja/Dhana Yoga 救援）。
**Kendra(角宫)**：1 / 4 / 7 / 10 = 事得势位。
**Trikona(三角宫)**：1 / 5 / 9 = 事得福位。

## 3. Chara Karaka(7K 主表) 速查

**KN Rao 7K 体系**：Sun~Saturn 共 7 颗按度数(0-30° 内)高→低排。承共享 KN Rao 硬约束 §3。

| Karaka | 中文 | Prashna 里的用途 |
|---|---|---|
| AK | 灵魂指示星 | 提问者深层驱动，问"要不要做" |
| AmK | 事业指示星 | 问职业/事业 |
| BK | 兄弟指示星 | 问兄弟姐妹关系 |
| MK | 母亲指示星 | 问母亲/家 |
| PiK | 父亲指示星 | 问父亲 |
| PK | 子女指示星 | 问怀孕/子女/学业 |
| GK | 障碍指示星 | 问疾病/敌人/诉讼(表达阻碍的一方) |
| DK | 配偶指示星 | 问婚姻/伴侣/合作 |

**禁引 8K/Rahu DK**（承 KN Rao 硬约束 §3）。

## 5. 使用注意

1. **一次一问**：一张 Prashna 盘只答一类问题；多类问题→拆分成多张盘或走 `question-taxonomy.md` 澄清流程。
2. **辅宫不喧宾**：辅宫用于加权/证据补充，主结论以主宫+主 Karaka 定。
3. **Karaka 优先级**：自然 Karaka > Chara Karaka(仅在问题明确对应时引)。
4. **表中未列**：走 `question-taxonomy.md` 归到最近类；实在无对应类，明确告知用户"此类问题超出 Prashna 标准判据范围"。
5. **禁做**：
   - ❌ 不看本表凭题面口感选宫
   - ❌ 引 8K/Rahu DK
   - ❌ 主宫堆超 2 个、辅宫堆超 2 个(过度重叠 = 结论稀释)
