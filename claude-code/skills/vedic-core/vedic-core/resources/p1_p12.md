# P1-P12 行星审计参数定义

> 来源：KN Rao体系(Parashari)
> 用途：core在审计每颗行星时参考此框架，按P1→P12逐层推导
> 面向用户输出时，必须翻译标签+附解释

## P1 角色身份 [关键权重]

行星的"职务"由它管理的宫位决定：

```
Core-Driver (1宫主) = 忠诚者
  → 永远服务盘主，力量越强越好

Yogakaraka (同掌吉凶宫) = CEO
  → 同时管三角宫和角宫，最高价值

Faithful (5/9宫主) = 盟友
  → 三角宫主，天然吉利

Trader (2/4/7/10宫主) = 交易者
  → Kendradhipati法则：吉星管角宫→变怠工，凶星管角宫→变自律

Growth-Hacker (3/6/11宫主) = 竞争者
  → 带毒增长，力量越强副作用越大

Destroyer (8/12宫主) = 清理者
  → 擅长结束和清理，科研/审计/海外有价值
```

**复合身份（面向用户的翻译）**：
- Soft-Betrayal = "温和错位"（吉星担凶职）→ 好心但用错了地方
- Hard-Construction = "高压建设者"（凶星担吉职）→ 苦劳出成果
- Growth-Hacker = "成长型竞争者" → 越用越强但有内耗

### P1.3 纹理审计（必须标注，不可省略）

检测自然属性与职能的冲突，这决定了执行过程的"体感"：

```
Soft-Betrayal（欺骗性风险）：
  触发条件：自然吉星（Jupiter/Venus/Mercury/Moon）担任GH或Destroyer角色
  典型案例：
    Jupiter掌6 = 过度扩张导致崩盘
    Venus掌8 = 沉溺感官导致毁灭
    Jupiter掌12 = 无节制的精神追求导致现实脱节
  审计要求：分析中必须标注"欺骗性风险"
  人话翻译："这颗星看着友善，但它管的事不友善"
  禁止：美化为"转化的力量""深层的智慧"

Hard-Construction（高压红利）：
  触发条件：自然凶星（Saturn/Mars/Rahu/Ketu）担任Core-Driver或Yogakaraka角色
  典型案例：
    Saturn为Yogakaraka = 极度自律出成果
    Mars为Core-Driver = 过程痛苦但结果是真的
  审计要求：分析中必须标注"高压红利"
  人话翻译："过程苦但结果是真的"
  禁止：淡化过程的痛苦，不能只说结果好
```

### P1.4 Maraka警示（健康分析时必须挂载）

```
触发条件：行星掌管2宫或7宫

规则：
  健康/寿命/重大意外分析时 → 挂载[Maraka]标签
    含义：该星在健康相关的Dasha期间有"终止"信号
    人话："这颗星管的领域跟你的身体安全有关联"

  事业/财富/家庭分析时 → 维持Trader身份
    不挂载Maraka标签
    人话："赚钱和谈合作的时候它就是个正常的执行者"

语境切换硬约束：
  同一颗星在不同分析板块中身份不同
  例：Venus管2宫+7宫
    健康板块 → [Maraka]"需关注Venus大运期间的健康波动"
    财富板块 → [Trader]"稳定的收入管理者"
    婚姻板块 → [Trader+7宫]"伴侣的品质由Venus状态决定"
  禁止在事业分析中提到"杀手""终止"等健康语境的标签
```

## P2 健康度 [重要权重]

```
燃烧(Combustion)临界距离：
  Moon: 12° | Mars: 17° | Mercury: 14°(逆行12°)
  Jupiter: 11° | Venus: 10°(逆行8°) | Saturn: 15°
  Cazimi (<1°): 仍受损（KN Rao观点）
  延迟补偿：Venus/Saturn燃烧解除后有强爆发

逆行：
  吉星逆行 = 深钻型（深度思考但表达困难）
  凶星逆行 = 回马枪（旧问题反复）
  KN Rao修正：落陷+逆行 → 视同入旺效果
              入旺+逆行 → 视同落陷效果

行星战争(Graha Yuddha)：
  两星距离 < 1°（Sun/Moon/Rahu/Ketu除外）
  胜者 = 经度小者（差 < 0.1° 时比Shadbala）
  败者 → 职能封锁转内耗，掌管宫位事务严重受损

Vargottama：
  D1和D9落在同一星座 = 健康度×150%

Dig Bala（方向力量）：
  Jupiter/Mercury → 1宫最强
  Moon/Venus → 4宫最强
  Saturn → 7宫最强
  Sun/Mars → 10宫最强
  行星落在最强宫位 → 获得Dig Bala加成（主场作战）
```

## P4 资源SAV(掌宫) [关键权重]

行星管理的宫位有多少资源：
```
SAV > 32 = 溢出（资源充裕，自动运转）
SAV 26-32 = 平稳（正常运作）
SAV 20-25 = 高阻（需要额外努力）
SAV < 20 = 匮乏（几乎无法运作）

特殊：凶宫(6/8/12) SAV高 = 反转效果
  → 6宫SAV高 = 强大的竞争力/健康恢复力
  → 8宫SAV高 = 意外收益/遗产/保险
```

## P5 路况(落宫性质) [重要权重]

行星落在什么类型的宫位：
```
吉路（1/4/5/7/9/10宫）= 损耗低，发挥顺畅
6宫 = 50%效率（竞争/疾病/服务环境）
8宫 = 30%效率（变化/危机/隐藏）
12宫 = 20%效率（损耗/隔离/灵性）

VRY（Viparita Raja Yoga）= 逆境王格
  基础条件：6/8/12宫主落入其他凶宫(6/8/12)
  分类：
    Harsha Yoga (6宫主入6/8/12) = 通过对手失误获利
    Sarala Yoga (8宫主入6/8/12) = 通过危机/继承获得地位
    Vimala Yoga (12宫主入6/8/12) = 低支出或在别人亏损的行业获利
```

**VRY孤立性判定（必须检查，不可跳过）**：
```
VRY不是只看宫主落宫就够了。必须检查该星是否被"污染"：

孤立性条件（决定VRY是否真的生效）：
  与吉宫主(1/5/9/10宫主)同宫且距离>10° → 弱连接
    VRY有效，但附注"福祸并存"
  与吉宫主同宫且距离<5° → 强融合
    VRY被稀释，转为"吉凶混杂"（按P3.2合相处理）
  与吉宫主精确合相<1° → VRY失效
    完全按该吉宫主的资源场分析

强度条件：
  1宫主(Lagna Lord)必须Shadbala>=100%，VRY才可判定为"逆境崛起"
  如果1宫主弱(Shadbala<100%)，VRY降级为"勉强求生"而非"逆境王者"

人话翻译：
  完整VRY = "你有在烂环境里翻盘的天赋"
  被稀释VRY = "你有一些韧性，但不是逆境型选手"
  VRY失效 = "不具备逆境翻盘特质"
```

## P6 出口SAV(落宫) [关键权重]

行星实际所在宫位的资源：
```
SAV > 32 = 超载（环境优越）
SAV 26-32 = 顺风
SAV 20-25 = 逆风
SAV < 20 = 崩塔
```

### P6.2 路况适配度（SAV+BAV交叉判定，必须执行）
```
高SAV + 高BAV = "瞬间超车"（环境好+个人强 → 最佳发挥）
低SAV + 高BAV = "乱世出英雄"（环境差+个人强 → 废土逆袭）
高SAV + 低BAV = "限速封路"（环境好+个人弱 → 浪费机会）
低SAV + 低BAV = "结构性毁灭"（环境差+个人弱 → 无法运作）

BAV阈值：
  BAV 5-8 = 高BAV（高效内驱）
  BAV 3-4 = 中等（维持）
  BAV 0-2 = 低BAV（失能）
```

## P12 格局Yogas [重要权重]

```
主要格局：
  Dharma-Karma Yoga (9-10宫主互动) = 系统溢价
    → 事业有使命感，社会认可度高
  
  Dhana Yoga (2-11宫主互动) = 倍增效应
    → 财富积累能力强
  
  Raja Yoga (三角宫主+角宫主合相/互视) = 王者格局
    → 权力/地位/影响力
  
  Viparita Raja Yoga (6/8/12宫主互溶) = 逆境王
    → 在困难中崛起

条件修正：
  参与星受损(燃烧/落陷/弱Shadbala) → 增益 × 0.2
  参与星强健 → 增益全额兑现
```

## 冲突仲裁规则

```
当多个P参数给出矛盾信号时：

1. P1清理者 + P7入旺 = "带毒高价值资产"
   → 禁止说"逢凶化吉"
   → 正确说法："在[领域]有很强的清理/转化能力，但过程有代价"

2. P5凶宫 + P6 BAV高 = "乱世出英雄"
   → 环境不好但个人能力强

3. P1吉 + P2受损 = "空有雄心无着力点"
   → 有好的角色但执行力受限

4. 多因素矛盾 → 不强行解释，标注"信号混合"
```
